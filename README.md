# Weather App – React Project
A modern weather application built with React + Vite that provides real-time weather data, 5-day forecast, air quality information, smart tips, and customizable settings.
## Live Demo
https://serafymstarobrianskyi.github.io/weather-react-app
## Tech Stack
- React
- Vite
- JavaScript
- CSS
- OpenWeather API

## Project Structure

```
src/
 ├── api/              # API calls
 ├── components/       # Reusable UI components
 ├── helpers/          # Utility functions (icons, tips, AQI)
 ├── pages/            # Application pages
 ├── store/            # Local storage + state logic
 ├── styles/           # Global styles
 ├── App.jsx
 └── main.jsx
```

## Scripts

- `npm install` - Install dependencies
- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run preview` - Preview production build
